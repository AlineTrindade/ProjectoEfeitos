Comandos:
Ls ---> Ver os ficheiros ou pastas que estão dentro de um pasta 
 
Cd (nome da pasta) ---> Entrar na Pasta

Python Efeitos.py ---> Executar o arquivo

$PWD ---> ver o caminho feito na Pasta

.\.venv\Scripts\Activate.ps1 ---> Ambiente virtual ativo

git init ---> criar repositório 

(comando que é para ser escrito no Workspace)

"terminal.integrated.defaultProfile.windows": "Command Prompt" (O profile não vai aparecer pois isso e exclusivo da PowerShell)                                   "PowerShell"

Depois e muito importante "salvar" o arquivo.
					      
> Após inserir um ficheiro novo em vscode , abrir terminal e por os códigos git add . , depois git commit -m 'escreve a modificação' , depois git status , depois git push , depois dá refresh no GitHub
> Após modificações em ficheiro existente em vscode , abrir terminal e por o código git commit -am 'escreve a modificação' , depois git status , depois git push , depois dá refresh no GitHub
> Para atualizar meu github, abro o terminal no ficheiro ProjetoEfeitos e escrevo o código git pull
